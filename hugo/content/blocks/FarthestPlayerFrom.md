Returns the farthest alive **Player** from a provided position. Can be filtered using a **TeamId**.  
  
_Note: If no players are alive when this block is called, the returned **Player** will be invalid._
### Inputs
1. **Vector**
    **(Position)**
2. **TeamId**
    **_Optional_**
### Output
-   **Player**
